## 1. Project Concept
SmartBacklog is a professional task management tool that integrates a **Local LLM (Large Language Model)** to assist Product Owners. It automates the generation of technical specifications while ensuring total data privacy.

## 2. Methodology Applied
- **Scrum Framework:** Iterative development across 2 Sprints.
- **MoSCoW Prioritization:** All "Must have" and "Should have" features are implemented.
- **Data Modeling:** Full CRUD lifecycle with persistent JSON storage.
- **Prompt Engineering:** Implementation of an "Expert Agile Coach" system prompt.
