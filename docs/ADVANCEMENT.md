# Advancement Report: SmartBacklog
**Project:** The intelligent assistant for Agile product management  
**Date:** March 24, 2026  
**Status:** Planning & Design Phase (Sprint 1 Initialized)

---

## 1. Project Concept & Methodology
The objective is to develop a web-based Kanban application enhanced by AI to automate Product Owner tasks. We are adopting the **Agile Scrum** methodology as outlined in the course, focusing on iterative delivery across two main Sprints.

## 2. Structural Breakdown (Epics)
Based on the project requirements, we have defined 3 primary Epics:
*   **Epic 1: Workspace Infrastructure:** Development of the Kanban board and column management.
*   **Epic 2: Ticket Management:** All CRUD operations (Create, Read, Update, Delete) for User Stories.
*   **Epic 3: AI Intelligence (Smart Features):** Integration of LLM (GPT-4o/Mistral) for automation.

## 3. Sprint Planning
Following the 2-week sprint cycle recommendation:

| Sprint | Objective | US Included | Status |
| :--- | :--- | :--- | :--- |
| **Sprint 1** | **The App Core (MVP):** Functional Kanban board and ticket lifecycle. | US1, US2 | In Progress |
| **Sprint 2** | **AI Integration:** Implementing Prompt Engineering and AI analysis. | US3, US4, US5 | Planned |

## 4. Technical Stack Advancement
*   **Frontend:** React with Tailwind CSS (Selected for modern UI/UX).
*   **Backend:** Node.js (Express) to handle API requests and AI orchestration.
*   **AI Side:** OpenAI API (GPT-4o) integration planned with a dedicated "System Prompt" to act as an Agile Coach.
